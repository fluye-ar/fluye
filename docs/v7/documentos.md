# Doors 7 - Documentos

> NOTA: Esta documentación se encuentra en construcción.

---

## ¿Qué son los Documentos?

Los documentos son conjuntos de datos almacenados en la base de datos de Doors que funcionan como instancias de formularios guardados en carpetas.

## Administrar Documentos

### Crear Documentos
Método `Folder.DocumentsNew` para crear un nuevo documento en una carpeta.

### Abrir Documentos
Diferentes modos de apertura: Tabla, Calendario.

### Copiar Documentos
Copia preservando propiedades y adjuntos.

### Mover Documentos
Movimiento entre carpetas del mismo tipo de formulario.

### Eliminar Documentos
Eliminación con opción de papelera de reciclaje para recuperación.

## Propiedades de un Documento

### Solapa General
Datos generales del documento.

### Solapa Campos
Valores asignados a cada campo del formulario.

### Log de Campos
Histórico de modificaciones del documento.

### Permisos

| Permiso | Código | Descripción |
|---------|--------|-------------|
| Leer | read | Permiso de lectura |
| Modificar | modify | Permiso de modificación |
| Borrar | delete | Permiso de eliminación |
| Administrar | admin | Permiso de administración |
