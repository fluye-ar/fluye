# Doors 7 - ASP Controls + Generic

> NOTA: Esta documentación se encuentra en construcción.

---

## Introducción

Los controles sirven para darle un aspecto personalizado a un documento de cierto formulario. Con ellos podemos organizar la página, configurándolos según las necesidades que se tengan.

El archivo `Generic.asp` / `Generic3.asp` construye interfaces de usuario basadas en controles, valida permisos, procesa valores, asigna campos y captura errores.
